MinimaxAdaptiveControl.dare — Method
dare(A, B, Q, R)

Compute X, the solution to the discrete-time algebraic Riccati equation, defined as A'XA - X - (A'XB)(B'XB + R)^-1(B'XA) + Q = 0, where Q>=0.

This version relaxes the requirement that R>0, rather R = [R1 0;0 R2] where R1 >0, R2 < 0. This is a usuful formulation for H_∞ synthesis.

Algorithm taken from: Laub, "A Schur Method for Solving Algebraic Riccati Equations." Relaxes the positive definiteness of R, useful for robust control.

http://dspace.mit.edu/bitstream/handle/1721.1/1301/R-0859-05666488.pdf

Implementation stolen from: ControlSystems.jl
